
## API de gerenciamento de arquivos Excel

Nesse sistema é possível subir arquivos excel, listar e remover dados.

Linguagem utilizada: PHP 
Framework: Laravel 5.5

## Instalação

- Execute o comando git clone https://github.com/ramonalves/react_client_benefits
- Execute o composer install para instalar as dependencias do projeto
- Edite o arquivo de configuração .env e coloque suas informações do banco de dados
- Crie o database (create database teste_fullstack) no MySQL
- Execute o comando php artisan migrate para criar as tabelas do banco de dados
- Execute o comando php artisan serve para iniciar o servidor
- Baixe e rode o repositório client (https://github.com/ramonalves/react_client_benefits)
